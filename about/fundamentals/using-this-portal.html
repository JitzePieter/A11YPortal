To Top

The baseline you should not drop below

The guidelines are best practices that can easily be tested with specific criteria which are not subjective and are technologically possible to achieve with current assistive technology on desktop and mobile devices.

Not following these guidelines means that some people with disabilities will find it difficult or impossible to use the web or native apps.

A11Y Fundamentals

  • Include people with all kind of disabilities.
  • Follow the legislation and guidelines to improve A11Y.

HTML WAI ARIA

  • Name, State, Role and Value.
  • Deliver the appropriate semantic information.

Universal Design

  • Designing all products to be accessible by everyone.
  • Allow flexibility, be simple, intuitive and minimize errors.

Mobile Web A11Y

  • Responsive design resembles low vision design.
  • Touch targets must be large enough to touch accurately.
